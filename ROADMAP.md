# Roadmap

This document outlines the future direction for Scion. Priorities may shift based on community feedback.

## Near-term

- **Matrix channel adapter** — add support for the Matrix protocol
- **Improved test coverage** — increase coverage for channel adapters, tools, and processing pipelines
- **Plugin system** — formalize the tool and channel registration APIs for third-party extensions
- **Persistent task storage** — back task management with database storage for cross-session continuity

## Mid-term

- **Web UI dashboard** — standalone admin panel for monitoring agent state, sessions, and logs
- **Multi-tenant support** — run multiple agent identities from a single deployment
- **Webhook channel** — generic inbound/outbound webhook adapter for custom integrations
- **Evaluation framework** — automated benchmarks for agent response quality and tool use accuracy
- **Voice conversations** — real-time voice input/output via channels that support it

## Long-term

- **Federated agents** — agent-to-agent communication across deployments
- **Fine-tuning pipeline** — use conversation history to fine-tune specialized models
- **Mobile companion app** — lightweight mobile client for direct agent interaction
