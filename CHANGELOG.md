# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-16

### Added

- Multi-channel messaging gateway (Slack, Telegram, Discord, WhatsApp, Google Chat, Signal, iMessage)
- Unified channel adapter interface with platform-native formatting
- 30+ built-in tools (file ops, shell/PTY, web, browser, task management, cron, TTS)
- Soul system with IDENTITY.md, SOUL.md, and USER.md personality files
- Multiple agent loop patterns (Ralph, Kimi, Swarm, Team)
- Memory system with semantic recall and working memory
- 11 pre-compiled skill workflows from Mermaid flowcharts
- CronService for agent-managed scheduled tasks
- HeartbeatService for proactive working memory monitoring
- Reflection workflow for pattern extraction from conversations
- Input/output processing pipelines (PII, prompt injection, adversarial detection)
- Media understanding pipeline (images, audio, video transcription)
- MCP client for external tool integration
- CLI for agent management
- Docker support
